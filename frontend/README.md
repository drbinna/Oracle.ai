# Frontend

This frontend is the operator and demo layer for Oracle.ai.

## Purpose

- visualize the agent organization
- surface rubric and evaluation results
- present the system cleanly during demos

## Run

```bash
npm install
npm run dev
```

## Recommended next steps

- connect to `http://localhost:8000/api/*`
- add charts for reward curves and model comparisons
- show filing-level analysis with citations
